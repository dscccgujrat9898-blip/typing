# DS Typing Trainer

Desktop typing trainer for English, Hindi and Gujarati with:

- Multi-user profiles (local)
- Practice, Speed Test and Exam Mode
- Flexible timing (pre-set and custom)
- History and HTML certificate generation
- Unit converter tool
- 100% offline, Python + Tkinter only

## Project Structure

- `main.py` – main application script
- `data/app_data.json` – user profiles and history (auto-created)
- `certificates/` – generated certificate HTML files
- `assets/fonts/` – place your official Hindi / Gujarati fonts here if needed
- `assets/images/` – reserved for keyboard layout images etc.

## Running the App (Developer Mode)

1. Install Python 3 (from python.org).
2. Open a terminal / command prompt in this folder.
3. Run:

```bash
python main.py
